Optional Exercise 1 Warmup Set 1 Set 2 Set 3 Set 4
Optional Exercise 2 Warmup Set 1 Set 2 Set 3 Set 4

MR10 - Complete as many reps as possible, but no more than 10

Extra Volume Squats- Add 5 lbs, then perform 5 sets x 3 reps each set with 60 seconds rest between each sets

If you were not able to complete a minimum of 8 reps on the MR10 set, reduce max entered by 2.5% moving forward

Complete the 5 sets of 3 reps regardless even if you did perform less than 8 reps on the MR10 set

Deadlift Variation Warmup Set 1 Set 2 Set 3 Set 4
Optional Exercise 1 Warmup Set 1 Set 2 Set 3 Set 4
Optional Exercise 2 Warmup Set 1 Set 2 Set 3 Set 4

MR10 - Complete as many reps as possible, but no more than 10

Back Off Squats - Reduce weight by 10 lbs, then proceed with the following:

If you completed 10 reps, perform 10 sets of 3 reps with 60 seconds rest between sets

If you completed 8-9 reps, perform 8 sets of 3 reps with 60 seconds rest between sets

If you completed 7 reps, perform 5 sets of 3 reps with 60 seconds rest between sets

If you completed less than 7 reps, skip back off sets entirely and reduce your 1 rep max by at least 2.5% for the following week

For Week 6 You Have 3 Options

1. Use calculated 1RM from last week's 1-4 rep set and start the next cycle.

2. Use projected max for next cycle, but take a deload week by repeating week 1 but skip the last upper body workout.

3. Take this 6th week to find your 1RM. Then either deload or start new cycle.
